A script is provided, "mvsctl" that controls the behavior of the server.

Example usage: ./mvsctl [option]

start: starts the server on 127.0.0.1:8080/

stop: stops the server

devel: start the server in development mode on 127.0.0.1:8080/

install: Installs all required libraries and launches tool to select the 
	 directory where the movies are located

build: creates an install for the server

change: launches a tool to change the location of the movies
